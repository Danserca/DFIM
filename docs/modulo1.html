Lección 2. Importancia de las estadísticas de calidad de Desplazamiento Forzado Interno

El desplazamiento forzado es un fenómeno persistente que plantea desafíos humanitarios importantes.

Se estima que a fines de 2022, 62.5 millones de personas fueron desplazadas por la fuerza en todo el mundo. Aunque solo representa 0.8 por ciento de la población de ese año, las personas desplazadas por la fuerza enfrentan numerosos desafíos sociales y económicos, incluida la pérdida de bienes y medios de vida, y necesitan de políticas públicas orientadas a la reconstrución de sus vidas.

El desarrollo de políticas destinadas a mejorar las condiciones socioeconómicas de las personas desplazadas internas requiere estadísticas confiables, comparables y oportunas sobre estas poblaciones.

Los datos también son necesarios para mejorar el uso de los recursos y las capacidades a fin de atender mejor las necesidades de estas poblaciones vulnerables. Las estadísticas producidas a nivel nacional son las más eficaces para fundamentar la respuesta nacional.

Lección 4. Principales fuentes de datos

Las IRIS presentan cuatro fuentes de datos principales que pueden utilizarse para producir estadísticas oficiales sobre personas en situación de desplazamiento forzado interno.

Además de las fuentes tradicionales conocidas (censo, encuestas y registros administrativos), consideran la integración de datos como una forma de combinar una variedad de fuentes de datos para mejorar la calidad de las estadísticas.

Cada fuente de datos tiene sus propias ventajas y limitaciones. Las IRIS contemplan una serie de enfoques y adaptaciones, como la inclusión de preguntas o módulos específicos, que se pueden aplicar a cada fuente de datos para ayudar a identificar grupos de población relevantes.
